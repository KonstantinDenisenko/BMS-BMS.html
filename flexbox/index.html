Свойства flexbox для контейнера

  • display: flex; display: inline-flex; - Активация flexbox (блочным или линейным)
  • 1)flex-direction: row; - элементы УКЛАДЫВАЮТСЯ в ряд слева направо
  • 2)flex-direction: row-reverse; - элементы УКЛАДЫВАЮТСЯ в ряд справа налево
  • 3)flex-direction: column; - элементы укладываются в колонку сверху вниз
  • 4)flex-direction: column-reverse; - элементы укладываются в колонку снизу вверх
  • 5)flex-wrap: wrap; - элементы отображаются в несколько рядов, если потребуется, слева направо и сверху вниз. По умолчанию (flex-wrap: nowrap; - элементы сужаются, чтобы уместиться в ширину flex-контейнера)
  • 6)flex-wrap: wrap-reverse; - элементы отображаются в несколько рядов, если потребуется, слева направо и снизу вверх. По умолчанию (flex-wrap: nowrap; - элементы сужаются, чтобы уместиться в ширину flex-контейнера)
  • flex-flow - Это свойство – сокращённая запись для свойств flex-direction и flex-wrap. Пример: flex-flow: row-reverse wrap-reverse;
  • 7)justify-content: flex-start; - элементы ВЫРАВНИВАЮТСЯ по левой стороне flex-контейнера в контексте
  • 8)justify-content: flex-end; - элементы ВЫРАВНИВАЮТСЯ по правой стороне flex-контейнера в контексте
  • 9)justify-content: center; - элементы выравниваются по центру flex-контейнера
  • 10)justify-content: space-between; - элементы отображаются с равным интервалом между ними, а первый и последний flex-элементы выравниваются по краям flex-контейнера
  • 11)justify-content: space-around; - элементы отображаются с равным интервалом вокруг каждого flex-элемента, включая первый и последний flex-элементы
  • 12)align-items: stretch; - элементы заполняют всю высоту (или ширину) от поперечного начала к поперечному концу flex-контейнера. ЭТО ЗНАЧЕНИЕ СТОИТ ПО УЛОЛЧАНИЮ
  • 13)align-items: flex-start; - элементы прижимаются к поперечному началу flex-контейнера
  • 14)align-items: flex-end; - элементы прижимаются к поперечному концу flex-контейнера
  • 15)align-items: center; - элементы укладываются по центру поперечной оси flex-контейнера
  • 16)align-items: baseline; - элементы выравниваются по базовой линии
  • 17)align-content: stretch; - элементы отображаются с распределённым пространством после каждого ряда flex-элементов
  • 18)align-content: flex-start; - элементы прижаты к поперечному началу flex-контейнера
  • 19)align-content: flex-end; - элементы прижаты к поперечному концу flex-контейнера
  • 20)align-content: center; - элементов укладываются по центру поперечной оси flex-контейнера
  • 21)align-content: space-between; - элементов отображаются с равным интервалом между ними, а первый и последний ряд выравнивается по краям flex-контейнера
  • 22)align-content: space-around; - элементы отображаются с равным интервалом вокруг каждого ряда flex-элементов

Свойства flexbox для элемента

  • 1)order: <целое число>; - элементы могут быть упорядочены при помощи этого простого свойства, без изменения структуры HTML-кода
  • 2)flex-grow: <число>; - задаёт элементам размер относительно друг друга, если число одинаковое, тогда и элементы будут одного размера
  • 3)flex-shrink: <число>; - указывает фактор гибкого растягивания, который определяет, насколько будет сужен flex-элемент относительно оставшихся flex-элементов во flex-контейнере при распределении отрицательного свободного пространства (отрицательные значения игнорируются)Значение по умолчанию: 1
  • 4)flex-basis: <ширина>px; - Это свойство принимает те же самые значения, что width and height, и указывает исходный главный размер flex-элемента
  • flex - Это свойство сокращённая запись для свойств flex-grow, flex-shrink и flex-basis. Наряду с другими значениями, его можно установить также в auto (1 1 auto) и none (0 0 auto)Пример: flex: -1 0 200px; flex: -1 0 auto; По умолчанию - flex: 0 1 auto;
  • 5)align-self: auto | flex-start | flex-end | center | baseline | stretch; - переопределяет выравнивание по умолчанию (или то, что указано в align-items) для отдельных flex-элементов
